This is a [Vue.js](https://vuejs.org/) project for the alert event test app.

Some key points:

- TypeScript
- The latest version
- ESLint + Prettier setup


# Getting Started

## Run the development server:

```bash
npm run dev
# or
yarn dev
```

## Build project:

```bash
npm run build
# or
yarn build
```

## Run unit test:

```bash
npm run test:unit
# or
yarn test:unit
```
